# UC-COMP-9: Pick Flow for Block

**Actor:** Author | **Tier:** Core | **Workflow:** WF3 Author Story

## Precondition
A composition is open in the editor and at least one flow exists in the library.

## Trigger
Click "Flow" in AddBlockBar.

## Main Flow
1. Author clicks the "Flow" button in the AddBlockBar
2. System opens the FlowPickerModal showing available flows
3. System loads flows from GET /api/v2/flows
4. Author searches or filters the flow list
5. Author selects a flow
6. Modal closes, block added with flow_id reference
7. Block shows flow name and step count

## Alternate Flows
- **AF-1:** No flows available -- Modal shows an empty state with a link to create a flow
- **AF-2:** Cancel -- Author closes the modal; no block is added

## Postcondition
A flow block is appended to the composition with a reference to the selected flow.
