# UC-COMP-2: Create Composition

**Actor:** Author | **Tier:** Core | **Workflow:** WF3 Author Story

## Precondition
Author is on the Compositions tab with an active session.

## Trigger
Click "+ New" button on the composition list.

## Main Flow
1. Author clicks "+ New" button on the composition list
2. System displays a creation form with kind selector (story / tutorial / module)
3. Author enters a title (required) and optional description
4. Author selects the composition kind from dropdown
5. System validates input with CompositionCreateSchema
6. System sends POST /api/v2/compositions
7. New composition appears in the list with zero blocks

## Alternate Flows
- **AF-1:** Missing title -- Save button remains disabled until a title is provided
- **AF-2:** Duplicate title -- System allows duplicate titles (compositions identified by ID)

## Postcondition
A new composition exists in the library with zero blocks, ready for block authoring.
