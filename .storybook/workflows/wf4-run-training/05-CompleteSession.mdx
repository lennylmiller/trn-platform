# UC-COMP-11: Complete Presentation

**Actor:** User | **Tier:** Core | **Workflow:** WF4 Run Training

## Precondition
Trainee has navigated to the last block in a composition.

## Trigger
Reach last block and click Next.

## Main Flow
1. User reaches last block
2. System shows "Last block" indicator
3. Next button disabled or shows "Complete"
4. Session summary available
5. User can navigate back to review blocks

## Alternate Flows
- **AF-1:** Return to library -- Trainee clicks "Return to Library" and is taken back to the composition list
- **AF-2:** Restart session -- Trainee clicks "Start New Session" to re-run the same composition

## Postcondition
The training session is marked as complete; trainee sees a summary of what was covered.
