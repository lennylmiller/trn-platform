# UC-COMP-11: Present (narrative focus)

**Actor:** User | **Tier:** Core | **Workflow:** WF4 Run Training

## Precondition
A composition has been selected and presentation mode is active.

## Trigger
Enter presentation mode.

## Main Flow
1. Composition loads with all blocks
2. First narrative block displays story content
3. Heading shown if set
4. Presenter notes available (collapsible)
5. User clicks Next to advance
6. Progress shows "Block N of M"
7. Narrative blocks show markdown-formatted prose

## Alternate Flows
- **AF-1:** Non-narrative block encountered -- System renders the appropriate block type (flow or note) instead
- **AF-2:** First block -- "Previous" button is disabled

## Postcondition
Trainee has read through narrative blocks and understands the context for upcoming flow blocks.
