# UC-STEP-1: Browse Step Library

**Actor:** User | **Tier:** Core | **Workflow:** WF5 Manage Steps

## Precondition
Author has an active session and steps exist in the library.

## Trigger
Navigate to Steps tab.

## Main Flow
1. System loads steps from GET /api/v2/steps
2. Steps grouped by category
3. Category filter chips shown
4. Search field for text filter
5. Each step shows label, type chip (shell=blue, sql=orange, manual=purple), category, description

## Alternate Flows
- **AF-1:** Empty library -- System shows an empty state with a prompt to create a step
- **AF-2:** Filter yields no results -- System shows "No steps match your filter"

## Postcondition
Author sees the full step library with management actions available.
